Jaded

Community Patched Jade Engine

Jaded is a community patched version of Ubisoft's Jade engine with a number of bug fixes to improve stability and other general improvements. It's still very much a work-in-progress, so expect the odd hiccup and do get in touch if you have any issues!

Features

  • Modernized appearance
  • In-built FPS cap that can be toggled
  • Clipboard toggle for Material editor
  • Support for MSAA for OpenGL renderer
  • More verbose dialog when saving world
  • A "Run Engine (Game Mode)" in place of the "Run Engine (Speed)" option
  • And a truck load of bug fixes! See here for a complete list.
