Jaded

Community Patched Jade Engine

Jaded is a community patched version of Ubisoft's Jade engine with a number of bug fixes to improve stability and other general improvements. It's still very much a work-in-progress, so expect the odd hiccup. Be sure to report any problems to the GitHub repository.

Latest release can be downloaded here.
Documentation can be found here.

Features

  • Editor and game runtime in one executable
  • Modernized appearance
  • FPS cap that can be toggled
  • Clipboard toggle for Material editor
  • Support for MSAA for OpenGL renderer
  • QoL improvements
  • And a truck load of bug fixes! See here for a complete list.
